Root Cause

In routers/web/admin/admin.go, function prepareStartupProblemsAlert():

func prepareStartupProblemsAlert(ctx *context.Context) { if len(setting.StartupProblems) > 0 { content := setting.StartupProblems[0] // <-- only first problem taken if len(setting.StartupProblems) > 1 { content += fmt.Sprintf(" (and %d more)", len(setting.StartupProblems)-1) } ctx.Flash.Error(content, true) // <-- single flash, rest hidden } }

With 2 deprecations in StartupProblems (from [log].ACCESS and [log].ROUTER), only the first is shown. The second is silently suppressed with "(and 1 more)".
