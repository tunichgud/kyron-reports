Technical Reasoning

Architectural

Vikunja 2.3.0 changed the desktop login from an internal browser dialog to 'open system default browser'. The bug lives in the handoff Electron app → external browser → callback. A Playwright browser can simulate neither the Electron startup nor the handoff between two real browsers.

Semantic

The auth code is a single-use OAuth artifact (RFC 6749 Section 10.5). It is correctly removed from the URL after the token exchange. This is not a defect but intentional security design. A test asserting this correct behavior as a failure would be a false positive.

Scope

Kyron scope (project memo): 'UI bugs only (Playwright-compatible), no backend-only issues'. Desktop app + multi-browser handoff falls outside this scope.
